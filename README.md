# germanwords-backend

Backend for German Words

## Run Production Server Locally

```sh
SECRET_KEY='GET_DATA'
DEBUG='False'
ENVIRONMENT='PROD'
DATABASE_URL="GET_DATA"
```

* Command: `python src/manage.py runserver 8080`
